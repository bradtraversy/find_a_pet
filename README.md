# Find-A-Pet

> Vanilla JS app that uses the Petfinder API to search for pets via zipcode. Uses Parcel, Babel and fetch-jsonp

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:1234
parcel

# Build for production
parcel build
```
